Heading Tags (<h1> até <h6>)

As tags de cabeçalho, de <h1> até <h6>, são usadas para definir títulos, onde <h1> é o mais importante e <h6> o menos importante.

P

A tag <p> é usada para definir um parágrafo de texto.

MARK

A tag <mark> destaca o texto em amarelo, como se fosse um marcador.

SMALL

A tag <small> reduz o tamanho do texto, geralmente para notas de rodapé ou avisos.

I

A tag <i> aplica uma formatação de texto em itálico.

U

A tag <u> sublinha o texto.

STRONG

A tag <strong> enfatiza o texto, geralmente com negrito, indicando maior importância.

OL

A tag <ol> cria uma lista ordenada, onde os itens são numerados.

UL

A tag <ul> cria uma lista não ordenada, onde os itens têm marcadores.

LI

A tag <li> define um item em uma lista, seja ordenada (<ol>) ou não ordenada (<ul>).

A

A tag <a> é usada para criar links para outras páginas ou seções.

HR

A tag <hr> insere uma linha horizontal para dividir o conteúdo.

SUB

A tag <sub> exibe o texto como um subíndice.

SUP

A tag <sup> exibe o texto como um sobrescrito.

BLOCKQUOTE

A tag <blockquote> é usada para citações em bloco, recuando o texto.

FONT

A tag <font> permite definir o tamanho, a cor e o tipo de fonte do texto. Embora seja depreciada, ainda pode aparecer em códigos antigos.

DEL

A tag <del> risca o texto, indicando conteúdo removido ou substituído.

ABBR

A tag <abbr> define uma abreviação ou acrônimo. Quando passa o cursor, um título explicativo aparece, se definido.
